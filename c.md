**Header Boilerplate**  
```
// By Peter Pyne  
/* By Peter Pyne*/  
```
**Includes**
```
#include <stdio.h>
```
**Symbolic Constants**  
```
#define SOMETHING	0 /* explanation */  
```
**Structure Boilerplate**  
```
main()  
{  
	float x, y;  
	int a, b;  
}  
```
**Input**  
```
a = getchar()  
```
**Print**  
```
printf(“hello, world\n”);  
printf(“%3d”, a); //3 chars allocated  
printf(“%5.1f”,x); //5 chars, 1 dec place (eg 1.1)  
putchar(a)  
```
**For Shorthand**  
```
for (a = 300; a >=0; a -= 20)  
```
**Valid Expressions**
```
= (Assign)  
-=  
+=  
<=  
>=  
!=  
```
